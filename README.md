Project Overview
This project analyses retail sales data from a Superstore dataset to uncover key business insights related to sales performance, profit trends, customer segments, and regional performance. The dashboard is built using Power BI to enable interactive data exploration and decision-making.

Objective

Analyse overall sales, profit, and order trends.
Identify top-performing regions, categories, and products.
Understand customer segments and their contribution.
Provide actionable insights and recommendations.

Tools Used
Power BI

Dashboard Features,
Sales and Profit KPIs,
Region-wise and Category-wise analysis,
Monthly and yearly sales trends,
Top N products and customers,
Segment-wise performance,
Interactive filters (slicers) for:,
Region,
Category,
Segment,
Order Date,
Dataset Description.

The dataset contains the following key fields:
Order ID,
Order Date,
Ship Date,
Region,
Category,
Sub-Category,
Sales,
Profit,
Quantity,
Discount,
Customer Segment.

Key Insights

Certain regions contribute significantly higher sales compared to others.
-The technology category generates the highest profit margins.
Discounts impact profit negatively in some categories.
A small group of products accounts for a large portion of total sales.
